# 連絡帳管理システム 利用マニュアル

本書は生徒・担任教師が連絡帳管理システムを使用する方法を説明します。

---

## はじめに

### システム概要

連絡帳管理システムは、生徒が毎日の体調・メンタル状態・振り返りを記録し、担任がそれを確認してフィードバックを行うWebアプリケーションです。

### アクセス方法

- **URL**: http://localhost:8000（開発環境）または https://d2wk3j2pacp33b.cloudfront.net（本番環境）
- **推奨ブラウザ**: Google Chrome、Microsoft Edge、Firefox、Safari

### ログイン方法

1. URLにアクセス
2. メールアドレスとパスワードを入力
3. 「ログイン」ボタンをクリック

---

## 生徒向けマニュアル

### 1. 連絡帳の作成・提出

#### 手順

1. ログイン後、生徒ダッシュボードが表示されます
2. 「今日の連絡帳を書く」ボタンをクリック
3. 以下の項目を入力します：

**入力項目**:

| 項目 | 説明 | 入力例 |
|-----|------|--------|
| **体調** | 今日の体調を★1〜★5で選択 | ★4（良い） |
| **メンタル** | 今日の気分を★1〜★5で選択 | ★3（普通） |
| **今日の振り返り** | 今日あったこと、感じたことを自由に記入 | 「数学のテストで85点取れました！」 |

4. 「提出」ボタンをクリック
5. 「連絡帳を提出しました」と表示されれば成功です

#### 注意事項

- **1日1件まで**: 1日に作成できる連絡帳は1件のみです
- **編集可能**: 担任が既読にする前であれば編集できます
- **既読後は編集不可**: 担任が既読にした後は編集できません

---

### 2. 過去の連絡帳閲覧

#### 手順

1. 生徒ダッシュボードで「過去の記録」リンクをクリック
2. 過去の連絡帳一覧が表示されます
3. 日付、体調、メンタル、担任からの反応が確認できます

#### 表示内容

| 項目 | 説明 |
|-----|------|
| **日付** | 連絡帳を提出した日 |
| **体調** | ★1〜★5 |
| **メンタル** | ★1〜★5 |
| **振り返り** | 当日記入した内容 |
| **担任からの反応** | 担任が選択した反応（👍、📝など） |

---

### 3. 担任からの反応確認

担任が連絡帳を確認すると、以下のような反応が表示されます。

| 反応 | 意味 |
|-----|------|
| 👍 | よくできました！ |
| 📝 | 記録しました |
| 💪 | よく頑張りましたね |
| ✨ | すばらしい！ |
| 🤝 | 応援しています |
| ✅ | 確認しました |

## 担任向けマニュアル

### 1. 担任ダッシュボード

#### 概要

担任ダッシュボードは、担当クラスの生徒の連絡帳を一覧表示し、優先度順に確認できる画面です。

#### 画面構成

ダッシュボードは以下の7つのセクションに分類されています：

| セクション | 優先度 | 説明 |
|-----------|--------|------|
| **P0: 重要** | 最高 | 3日連続メンタル低下、メンタル★1（即時対応が必要） |
| **P1: 要注意** | 高 | メンタル★2（注意が必要） |
| **P1.5: 要対応タスク** | 高 | 既読済みで対応記録あり（保護者連絡、個別面談など） |
| **P2-1: 未提出** | 中 | 今日の連絡帳をまだ提出していない生徒 |
| **P2-2: 未読** | 中 | 提出済みだがまだ確認していない連絡帳 |
| **P2-3: 反応待ち** | 低 | 既読済みだが反応を選択していない連絡帳 |
| **P3: 完了済み** | 低 | 既読・反応済みの連絡帳 |

---

### 2. 連絡帳の確認・既読処理

#### 手順

1. 担任ダッシュボードで未読の連絡帳カードを確認
2. 「既読にする」ボタンをクリック
3. 以下の項目を選択します：

**生徒への反応**（生徒に表示される）:

| 選択肢 | 使用場面 |
|--------|---------|
| 👍 よくできました！ | 良い内容の連絡帳 |
| 📝 記録しました | 通常の記録 |
| 💪 よく頑張りましたね | 頑張りを認める |
| ✨ すばらしい！ | 特に良い内容 |
| 🤝 応援しています | 励ましが必要な場合 |
| ✅ 確認しました | 確認のみ |

**対応記録**（先生のみに表示）:

| 選択肢 | 使用場面 |
|--------|---------|
| 記録のみ | 特に対応不要 |
| 要フォローアップ | 後日確認が必要 |
| 緊急対応 | 即座の対応が必要 |
| 保護者連絡 | 保護者への連絡が必要 |
| 個別面談 | 生徒との面談が必要 |
| 学年会議で共有 | 学年全体で情報共有 |
| 見守り中 | 継続的な観察が必要 |

4. 「保存」ボタンをクリック

#### ショートカット機能

- **既読のみ**: 「既読にする（対応不要）」ボタンをクリックすると、反応・対応記録を選択せずに既読処理のみ実行
- **タスク化**: 「タスク化」ボタンをクリックすると、既読処理と同時に対応記録を設定

---

### 3. 担任メモの作成・共有

#### 概要

担任メモは生徒についての長期的な観察記録や引継ぎ情報を記録する機能です。

#### 種類

| 種類 | 説明 | 閲覧範囲 |
|-----|------|---------|
| **個人メモ** | 自分だけが見るメモ | 作成した担任のみ |
| **学年共有メモ** | 学年の担任全員で共有するメモ | 同じ学年の担任全員 |

#### 手順（個人メモ）

1. 担任ダッシュボードで生徒名をクリック
2. 生徒詳細ページが表示される
3. 「メモを追加」ボタンをクリック
4. メモ内容を入力
5. 「学年会議で共有」のチェックは**オフ**のまま
6. 「保存」ボタンをクリック

#### 手順（学年共有メモ）

1. 担任ダッシュボードで生徒名をクリック
2. 生徒詳細ページが表示される
3. 「メモを追加」ボタンをクリック
4. メモ内容を入力
5. 「学年会議で共有」のチェックを**オン**にする
6. 「保存」ボタンをクリック

#### 共有メモの既読処理

学年共有メモは、他の担任が作成したメモを既読処理できます。

1. 担任ダッシュボードで「学年共有アラート」セクションを確認
2. 未読の共有メモが表示される
3. 「既読にする」ボタンをクリック

---

### 4. 出席記録の入力

#### 手順

1. 担任ダッシュボードで「出席記録」セクションを確認
2. 各生徒の出席状況を選択

**出席状況の選択肢**:

| 選択肢 | 説明 |
|--------|------|
| ✅ 出席 | 通常登校 |
| ❌ 欠席 | 欠席（理由を選択） |
| ⏰ 遅刻 | 遅刻 |
| 🏃 早退 | 早退 |

**欠席理由**（欠席の場合のみ）:

| 選択肢 | 説明 |
|--------|------|
| 病気 | 風邪、インフルエンザなど |
| 家庭の事情 | 冠婚葬祭など |
| その他 | その他の理由 |

3. 入力後、自動保存される

## よくある質問（FAQ）

### 生徒向け

**Q1. 連絡帳を間違えて提出してしまいました。修正できますか？**

A1. 担任が既読にする前であれば、生徒ダッシュボードから「編集」ボタンをクリックして修正できます。既読後は修正できません。

**Q2. 1日に2回連絡帳を提出できますか？**

A2. いいえ、1日1件までです。システムの制約として、同じ日付で複数の連絡帳を作成することはできません。

**Q3. 体調やメンタルの★の意味がわかりません。**

A3. ★1〜★5で選択します。
- ★1: とても悪い
- ★2: 悪い
- ★3: 普通
- ★4: 良い
- ★5: とても良い

**Q4. 担任からの反応がまだ表示されません。**

A4. 担任が連絡帳を確認するまで反応は表示されません。通常、当日または翌日には確認されます。

---

### 担任向け

**Q1. 未読の連絡帳が多すぎて確認しきれません。優先順位はありますか？**

A1. はい、担任ダッシュボードは優先度順に表示されています。まず「P0: 重要」セクションから確認してください。3日連続メンタル低下やメンタル★1の生徒は即座の対応が必要です。

**Q2. 学年共有メモと個人メモの違いは何ですか？**

A2.
- **個人メモ**: 自分だけが見るメモ。他の担任には表示されません。
- **学年共有メモ**: 同じ学年の担任全員が見るメモ。学年会議で情報共有する際に使用します。

**Q3. 対応記録を間違えて選択してしまいました。修正できますか？**

A3. はい、生徒の連絡帳詳細ページから再度「既読にする」ボタンをクリックして、正しい対応記録を選択し直してください。

**Q4. 出席記録を間違えて入力しました。修正できますか？**

A4. はい、担任ダッシュボードの「出席記録」セクションで、正しい出席状況を選択し直してください。自動保存されます。

**Q5. 他のクラスの生徒の連絡帳も見れますか？**

A5. いいえ、担任は自分が担当するクラスの生徒の連絡帳のみ確認できます。これはプライバシー保護のための仕様です。
