## 生徒の操作

### ログイン情報
```
メールアドレス: student@example.com
パスワード: password123
```

### 操作手順

#### 1. ログイン
1. ブラウザで `https://d2wk3j2pacp33b.cloudfront.net` にアクセス
2. 「ログイン」ボタンをクリック
3. メールアドレス: `student@example.com` を入力
4. パスワード: `password123` を入力
5. 「ログイン」ボタンをクリック
6. ✅ **確認:** ダッシュボード（トップページ）が表示されることを確認

#### 2. 連絡帳の新規作成
1. トップページの「✏️ 新規作成」ボタンをクリック
2. または直接 `https://d2wk3j2pacp33b.cloudfront.net/diary/new/` にアクセス
3. **記載日:** 前日の日付を選択（自動で前日が設定されているはず）
4. **体調:** 「良い」を選択
5. **メンタル:** 「元気」を選択
6. **今日の振り返り:** 以下のテスト文を入力
   ```
   今日は数学のテストがありました。
   準備をしっかりしていたので、手応えを感じました。
   部活動では新しい技を練習しました。
   ```
7. 「📝 提出する」ボタンをクリック
8. ✅ **確認:** 「連絡帳を提出しました」というメッセージが表示されることを確認
9. ✅ **確認:** 一覧ページにリダイレクトされ、作成した連絡帳が表示されることを確認

#### 3. 提出済み連絡帳の確認
1. トップページの「📖 一覧」ボタンをクリック
2. または直接 `https://d2wk3j2pacp33b.cloudfront.net/diary/` にアクセス
3. ✅ **確認:** 自分が提出した連絡帳の一覧が表示されることを確認
4. 任意の連絡帳をクリックして詳細を表示
5. ✅ **確認:**
   - 記載日、体調、メンタル、振り返り内容が正しく表示されることを確認
   - 担任からの反応（👍 いいね等）が表示される場合があることを確認

#### 4. 連絡帳の編集（提出後24時間以内のみ）
1. 連絡帳一覧から最新の連絡帳をクリック
2. 「✏️ 編集」ボタンが表示されている場合はクリック
3. 振り返り内容を修正
4. 「📝 提出する」ボタンをクリック
5. ✅ **確認:** 「連絡帳を更新しました」というメッセージが表示されることを確認

#### 5. ログアウト
1. 画面右上のユーザー名をクリック
2. 「ログアウト」をクリック
3. ✅ **確認:** ログイン画面に戻ることを確認

## 担任の操作

### ログイン情報
```
メールアドレス: teacher@example.com
パスワード: password123
```

### 操作手順

#### 1. ログイン
1. ブラウザで `https://d2wk3j2pacp33b.cloudfront.net` にアクセス
2. 「ログイン」ボタンをクリック
3. メールアドレス: `teacher@example.com` を入力
4. パスワード: `password123` を入力
5. 「ログイン」ボタンをクリック
6. ✅ **確認:** 担任ダッシュボードが表示されることを確認

#### 2. 受信箱（Inbox）の確認
1. トップページ（ダッシュボード）に「📥 受信箱」セクションが表示されることを確認
2. ✅ **確認:** 以下のカードが表示されることを確認
   - 🔴 緊急対応が必要な生徒（体調・メンタルが「とても悪い」）
   - 🟡 注意が必要な生徒（体調・メンタルが「悪い」）
   - 📝 未読の連絡帳
   - ⚠️ 未提出の生徒（前日分）
3. 各カードの生徒名をクリックして連絡帳詳細に遷移できることを確認

#### 3. クラス全体の連絡帳確認
1. ナビゲーションバーの「📖 連絡帳一覧」をクリック
2. または直接 `https://d2wk3j2pacp33b.cloudfront.net/teacher/diary-list/` にアクセス
3. ✅ **確認:** 自分のクラスの生徒の連絡帳が一覧表示されることを確認
4. ✅ **確認:** 各生徒の最新の体調・メンタル状態が色分けされて表示されることを確認
   - 🔴 赤: とても悪い
   - 🟡 オレンジ: 悪い
   - ⚪ グレー: 普通
   - 🟢 緑: 良い
   - 🔵 青: とても良い

#### 4. 生徒への反応を追加
1. 連絡帳一覧から任意の生徒の連絡帳をクリック
2. 連絡帳詳細ページで「💬 反応・対応を記録」ボタンをクリック
3. **生徒に見える反応:** 「👍 いいね」を選択
4. **コメント（任意）:** 「よく頑張りましたね！」と入力
5. 「💾 保存」ボタンをクリック
6. ✅ **確認:** 「反応を記録しました」というメッセージが表示されることを確認
7. ✅ **確認:** 連絡帳詳細ページに戻り、追加した反応が表示されることを確認

#### 5. 内部対応記録の追加
1. 連絡帳詳細ページで「💬 反応・対応を記録」ボタンをクリック
2. **先生だけが見る対応:** 「⚠️ 要フォロー」を選択
3. **対応メモ（任意）:** 「明日個別に声をかける」と入力
4. **対応状況:** 「対応中」を選択
5. 「💾 保存」ボタンをクリック
6. ✅ **確認:** 対応記録が保存されることを確認

#### 6. 出席管理
1. ナビゲーションバーの「✓ 出席管理」をクリック
2. または直接 `https://d2wk3j2pacp33b.cloudfront.net/teacher/attendance/` にアクセス
3. ✅ **確認:** 自分のクラスの生徒一覧が表示されることを確認
4. 任意の生徒の出席状態を変更
   - 「欠席」をクリック → モーダルが開く
   - 欠席理由を選択（例: 体調不良）
   - 「保存」をクリック
5. ✅ **確認:** 出席状態が更新されることを確認

#### 7. ログアウト
1. 画面右上のユーザー名をクリック
2. 「ログアウト」をクリック
3. ✅ **確認:** ログイン画面に戻ることを確認

## トラブルシューティング

### ログインできない場合
1. メールアドレスとパスワードを正確に入力しているか確認
2. パスワードは **すべて小文字** で `password123` です
3. ブラウザのシークレットモード（プライベートモード）で試してみる
4. ブラウザのキャッシュをクリアして再度試す

### ページが表示されない場合
1. インターネット接続を確認
2. URLが正しいか確認: `https://d2wk3j2pacp33b.cloudfront.net`
3. ブラウザを再起動して再度アクセス

### エラーメッセージが表示された場合
1. エラーメッセージ全体をスクリーンショットで保存
2. エラーメッセージをコピー＆ペーストでテキストとして保存
3. どの操作でエラーが発生したかをメモ

### 操作がわからない場合
1. ナビゲーションバー（画面上部）から目的の機能を探す
2. ダッシュボード（トップページ）に戻って再度操作する
3. 不明な場合はスキップして次の操作に進む

## 補足情報

### システムの特徴
- **メール認証不要:** パスワードのみでログイン可能
- **ロールベース権限:** 各ユーザーは自分の役割に応じた機能のみアクセス可能

- このシステムには既に295名のユーザーと多数の連絡帳データが登録されています
- 自由に操作・編集して問題ありません（テスト環境のため）
