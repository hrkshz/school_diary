## 🎯 概要

kits.ioは、CSV/TSV/Excelファイルのインポート機能を提供するDjangoパッケージです。

**主要機能**:
- ✅ CSV/TSV/Excelファイルのインポート
- ✅ 文字コード自動検出（Shift-JIS等）
- ✅ 重複チェック・新規採番
- ✅ Djangoフォームベースのバリデーション
- ✅ 詳細なインポート履歴管理
- ✅ Django管理画面・CLIコマンド

## 🚀 クイックスタート

### 最小限のコード（3ステップ）

```python
from kits.io.importers import CSVImporter
from django.contrib.auth import get_user_model

User = get_user_model()

# 1. インポーター作成
importer = CSVImporter(
    model_name='library.Book',
    user=User.objects.first(),
)

# 2. インポート実行
history = importer.import_file('books.csv')

# 3. 結果確認
print(f"成功: {history.success_count}件")
```

詳しくは **[05_使い方ガイド.md](05_使い方ガイド.md)** を参照してください。

## 🔧 主要機能

### 1. ファイル形式対応

- CSV（カスタム区切り文字対応）
- TSV（タブ区切り）
- Excel（.xlsx、シート指定可能）

### 2. 文字コード検出

- 自動検出（chardet使用）
- UTF-8、Shift-JIS、EUC-JPなど

### 3. 重複処理戦略

- **skip**: 重複をスキップ
- **update**: 既存レコードを更新
- **renumber**: 新規採番（連番付加）
- **error**: エラーとして扱う

### 4. インポート履歴管理

- 統計情報（成功、失敗、スキップ、更新、新規採番）
- 成功率、処理時間
- 行ごとのエラー詳細
