1. **品質基準の確認**
   ```bash

   cat docs/kits/DOCUMENTATION_STANDARDS.md
   ```

2. **チェックリストの準備**
   ```bash

   cp docs/kits/IMPLEMENTATION_CHECKLIST_TEMPLATE.md \

   ```

3. **実装完了後の品質チェック**
   ```bash
   # 必ず実行
   python3 scripts/check_kits_docs_quality.py {パッケージ名}
   ```
