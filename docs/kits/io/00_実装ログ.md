## 実装概要

kits.ioパッケージの実装を完了しました。CSV/TSV/Excelファイルのインポート機能、バリデーション、重複チェック、新規採番、インポート履歴管理を提供します。

### 実装した機能

1. **データモデル**
   - ImportMapping: マッピング設定の管理
   - ImportHistory: インポート履歴の記録

2. **インポーター**
   - CSVImporter: CSV形式のインポート
   - TSVImporter: TSV形式のインポート
   - ExcelImporter: Excel形式のインポート

3. **バリデーター**
   - ImportValidator: Djangoフォームベースのバリデーション
   - CustomValidator: カスタムバリデーション

4. **管理機能**
   - Django管理画面
   - import_data管理コマンド

## 設計判断の記録

### 1. UUIDをPKに採用（ImportHistory）

**理由**: セキュリティ向上、インポートIDの予測防止

### 2. ステータス管理

**ステータス**: pending → processing → completed/failed/partial

**理由**: 処理の進捗追跡、エラーハンドリング

### 3. チャンク処理

**デフォルトサイズ**: 1000行

**理由**: 大量データでもメモリ効率的、トランザクション単位

### 4. マッピング設定の分離

**理由**: 再利用可能な設定、ImportHistoryとの1対多関係

### 5. 重複処理の柔軟性

**4戦略**: skip, update, renumber, error

**理由**: 過去課題（図書館システムのバーコードID重複）に対応
