## 基本的な使い方

### 最もシンプルな例

```python
from kits.io.importers import CSVImporter
from django.contrib.auth import get_user_model

User = get_user_model()

# 1. インポーターを作成
importer = CSVImporter(
    model_name='library.Book',  # アプリ名.モデル名
    user=User.objects.first(),  # 実行ユーザー
)

# 2. インポート実行
history = importer.import_file('/path/to/books.csv')

# 3. 結果を確認
print(f"成功: {history.success_count}件")
print(f"失敗: {history.failed_count}件")
print(f"成功率: {history.success_rate}%")
```

## CSVインポート

### 例1: シンプルなCSVインポート

**ファイル**: `books.csv`
```csv
title,author,isbn,price
Djangoの教科書,山田太郎,978-4-1234-5678-9,3000
Pythonクックブック,田中花子,978-4-9876-5432-1,3500
```

**コード**:
```python
from kits.io.importers import CSVImporter
from django.contrib.auth import get_user_model

User = get_user_model()

# インポーター作成
importer = CSVImporter(
    model_name='library.Book',
    user=User.objects.first(),
)

# インポート実行
history = importer.import_file('books.csv')

# 結果確認
if history.status == 'completed':
    print(f"✅ {history.success_count}件のインポートに成功しました")
else:
    print(f"⚠️ 一部失敗: 成功{history.success_count}件、失敗{history.failed_count}件")
```

### 例2: カラム名が日本語の場合（マッピング）

**ファイル**: `books_ja.csv`
```csv
タイトル,著者,ISBN,価格
Djangoの教科書,山田太郎,978-4-1234-5678-9,3000
```

**コード**:
```python
from kits.io.importers import CSVImporter
from django.contrib.auth import get_user_model

User = get_user_model()

# マッピングを指定
importer = CSVImporter(
    model_name='library.Book',
    mapping={
        'タイトル': 'title',
        '著者': 'author',
        'ISBN': 'isbn',
        '価格': 'price',
    },
    user=User.objects.first(),
)

history = importer.import_file('books_ja.csv')
```

### 例3: Shift-JIS エンコーディング

**Windowsで作成されたCSVファイル（Shift-JIS）**:

```python
from kits.io.importers import CSVImporter

# 方法1: 自動検出（推奨）
importer = CSVImporter(
    model_name='library.Book',
    auto_detect_encoding=True,  # 自動検出
    user=user,
)

# 方法2: 明示的に指定
importer = CSVImporter(
    model_name='library.Book',
    encoding='shift-jis',
    auto_detect_encoding=False,
    user=user,
)

history = importer.import_file('books_shift_jis.csv')
```

### 例4: カスタム区切り文字（セミコロン等）

**ファイル**: `books_semicolon.csv`
```csv
title;author;isbn
Book A;Author A;123
Book B;Author B;456
```

**コード**:
```python
from kits.io.importers import CSVImporter

importer = CSVImporter(
    model_name='library.Book',
    delimiter=';',  # セミコロン区切り
    user=user,
)

history = importer.import_file('books_semicolon.csv')
```

## Excelインポート

### 例6: Excelファイル（最初のシート）

**ファイル**: `books.xlsx`

| タイトル | 著者 | ISBN |
|---------|------|------|
| Book A  | Author A | 123 |
| Book B  | Author B | 456 |

**コード**:
```python
from kits.io.importers import ExcelImporter
```

## 重複処理

### 例9: 重複をスキップ（デフォルト）

```python
from kits.io.importers import CSVImporter

importer = CSVImporter(
    model_name='library.Book',
    unique_fields=['isbn'],  # ISBNで重複チェック
    duplicate_strategy='skip',  # 重複はスキップ
    user=user,
)

history = importer.import_file('books.csv')
print(f"スキップ: {history.skipped_count}件")
```

### 例10: 重複したら更新

```python
from kits.io.importers import CSVImporter

importer = CSVImporter(
    model_name='library.Book',
    unique_fields=['isbn'],
    duplicate_strategy='update',  # 重複したら既存レコードを更新
    user=user,
)

history = importer.import_file('books.csv')
print(f"更新: {history.updated_count}件")
```

### 例11: 重複したら新規採番（課題4対応）

**ユースケース**: 5つの学校図書館のデータを統合、バーコードIDが重複

```python
from kits.io.importers import ExcelImporter

# A校のデータ
importer = ExcelImporter(
    model_name='library.Book',
    mapping={'バーコードID': 'barcode_id', 'タイトル': 'title'},
    unique_fields=['barcode_id'],
    duplicate_strategy='renumber',  # 重複したら新規採番
    sheet_name='A校',
    user=user,
)

history_a = importer.import_file('schools.xlsx')
print(f"A校: 新規採番{history_a.renumbered_count}件")

# B校のデータ（同じ設定）
importer.sheet_name = 'B校'
history_b = importer.import_file('schools.xlsx')
print(f"B校: 新規採番{history_b.renumbered_count}件")

# 結果:
# A校: ABC-001 → ABC-001（そのまま）
# B校: ABC-001 → ABC-001_00001（新規採番）
```

### 例12: 重複をエラーとして扱う

```python
from kits.io.importers import CSVImporter

importer = CSVImporter(
    model_name='library.Book',
    unique_fields=['isbn'],
    duplicate_strategy='error',  # 重複時はエラー
    user=user,
)

history = importer.import_file('books.csv')
print(f"失敗（重複エラー）: {history.failed_count}件")

# エラー詳細を確認
for error in history.error_details:
    print(f"行{error['row']}: {error['error']}")
```

## マッピング設定

### 例13: マッピング設定を作成して再利用

**ステップ1**: マッピング設定を作成（1回だけ）

```python
from kits.io.models import ImportMapping

# マッピング設定を作成
mapping = ImportMapping.objects.create(
    code='book_import',
    name='図書インポート',
    description='図書館システム用の標準インポート設定',
    model_name='library.Book',
    field_mapping={
        'タイトル': 'title',
        '著者': 'author',
        'ISBN': 'isbn',
        '価格': 'price',
        '出版社': 'publisher',
    },
    unique_fields=['isbn'],
    duplicate_strategy='skip',
    is_active=True,
)

print(f"マッピング設定 '{mapping.code}' を作成しました")
```

**ステップ2**: マッピング設定を使用（以降は簡単）

```python
from kits.io.importers import CSVImporter

# コードを指定するだけ！
importer = CSVImporter(
    model_name='library.Book',
    mapping_code='book_import',  # 簡単
    user=user,
)

history = importer.import_file('books.csv')
```

**ステップ3**: 管理画面で編集

Django管理画面 → "Data Import/Export" → "インポートマッピング" から
`book_import`を編集できます。

## 管理コマンド

### 例14: コマンドラインからインポート

```bash
# 基本的な使い方
python manage.py import_data \
    books.csv \
    library.Book \
    --format=csv

# 詳細オプション付き
python manage.py import_data \
    books.csv \
    library.Book \
    --format=csv \
    --encoding=shift-jis \
    --duplicate-strategy=update \
    --mapping=book_import
```

### 例15: TSVインポート（コマンド）

```bash
python manage.py import_data \

    --format=tsv \
```

## Django Shell

### 例17: Django Shellでインポート

```bash
# Django Shell起動
python manage.py shell
```

```python
# Shell内で実行
from kits.io.importers import CSVImporter
from django.contrib.auth import get_user_model

User = get_user_model()
user = User.objects.first()

# インポート実行
```

### 実践例1: 野球部記録（課題3）

**要件**: 過去2年分の記録.xlsxを取り込む

```python
from kits.io.importers import ExcelImporter
from django.contrib.auth import get_user_model

User = get_user_model()
user = User.objects.first()

# マッピング設定作成
mapping = ImportMapping.objects.create(
    code='baseball_record',
    name='野球部記録インポート',
    model_name='baseball.Record',
    field_mapping={
        '日付': 'date',
        '選手名': 'player_name',
        '打数': 'at_bats',
        '安打': 'hits',
        '打率': 'batting_average',
    },
)

# 2022年度
importer = ExcelImporter(
    model_name='baseball.Record',
    mapping_code='baseball_record',
    sheet_name='2022年度',
    user=user,
)
history_2022 = importer.import_file('記録.xlsx')
print(f"2022年度: {history_2022.success_count}件")

# 2023年度
importer.sheet_name = '2023年度'
history_2023 = importer.import_file('記録.xlsx')
print(f"2023年度: {history_2023.success_count}件")
```

### 実践例2: 図書館システム（課題4）

**要件**:
- 5つの学校図書館データをTSVで受け取る
- バーコードIDが重複している場合は新規採番

```python
from kits.io.importers import TSVImporter
from kits.io.models import ImportMapping

# マッピング設定作成
mapping = ImportMapping.objects.create(
    code='library_import',
    name='図書館インポート',
    model_name='library.Book',
    field_mapping={
        'タイトル': 'title',
        '著者': 'author',
        'バーコードID': 'barcode_id',
    },
    unique_fields=['barcode_id'],
    duplicate_strategy='renumber',  # 重複時は新規採番
)

# 5つの学校データをインポート
schools = ['school_a', 'school_b', 'school_c', 'school_d', 'school_e']

for school in schools:
    importer = TSVImporter(
        model_name='library.Book',
        mapping_code='library_import',
        user=user,
    )

    history = importer.import_file(f'{school}_books.tsv')
    print(f"{school}: 成功{history.success_count}件、新規採番{history.renumbered_count}件")

# 結果例:
# school_a: 成功1000件、新規採番0件
# school_b: 成功950件、新規採番50件（重複）
# school_c: 成功980件、新規採番20件（重複）
# ...
```

### 実践例3: 定期的なデータ更新

**要件**: 毎月、取引先から送られてくるCSVデータを取り込む

```python
from kits.io.importers import CSVImporter
from kits.io.models import ImportMapping
import datetime

# マッピング設定（1回だけ作成）
mapping, created = ImportMapping.objects.get_or_create(
    code='monthly_sales',
    defaults={
        'name': '月次売上データ',
        'model_name': 'sales.SalesRecord',
        'field_mapping': {
            '商品コード': 'product_code',
            '数量': 'quantity',
            '金額': 'amount',
        },
        'unique_fields': ['product_code'],
        'duplicate_strategy': 'update',  # 既存データを更新
    }
)

# 毎月実行（Celeryタスク等で自動化）
def monthly_import():
    today = datetime.date.today()
    filename = f'sales_{today.year}{today.month:02d}.csv'

    importer = CSVImporter(
        model_name='sales.SalesRecord',
        mapping_code='monthly_sales',
        user=user,
    )

    history = importer.import_file(filename)

    # 管理者にメール通知
    if history.status == 'completed':
        send_mail(
            subject=f'[成功] {filename}のインポート完了',
            message=f'成功: {history.success_count}件',
            from_email='system@example.com',
            recipient_list=['admin@example.com'],
        )
    else:
        send_mail(
            subject=f'[失敗] {filename}のインポートエラー',
            message=f'失敗: {history.failed_count}件\n{history.error_message}',
            from_email='system@example.com',
            recipient_list=['admin@example.com'],
        )

# 実行
monthly_import()
```
