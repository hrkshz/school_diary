## 実装の学び

### 1. django-simple-historyのベストプラクティス

```python
# HistoricalRecords(inherit=True)で継承可能に
class AuditMixin(models.Model):
    history = HistoricalRecords(inherit=True)
    
    class Meta:
        abstract = True
```

### 2. TYPE_CHECKINGパターン

他のkitsと同様に、循環インポートを避けるためのパターンを採用。

### 3. Django Adminのread-only実装

```python
def has_add_permission(self, request):
    return False  # 追加不可

def has_change_permission(self, request, obj=None):
    return False  # 変更不可

def has_delete_permission(self, request, obj=None):
    return request.user.is_superuser  # スーパーユーザーのみ削除可
```

### 4. JSONFieldの活用

変更内容を柔軟に記録:
```python
changes = {
    "status": {"from": "draft", "to": "published"},
    "title": {"from": "Old", "new": "New"}
}
```
