# kits.reports - 概要と目的

**読了時間:** 約5分

---

## このパッケージは何？

`kits.reports`は、業務アプリケーションで必須となる**レポート機能・データ可視化機能**を提供する再利用可能なDjangoパッケージです。

複雑なデータを分かりやすく可視化し、PDF・Excel・CSV形式でエクスポートできます。

---

## なぜ必要なのか？

### 解決する課題

**課題1: 残業管理システムのレポート**（過去課題1）
- **問題:** 社員ごとの残業時間を集計し、部門別グラフで可視化したい
- **解決:**
  - 月次残業時間レポートをPDF出力
  - 部門別棒グラフ・個人別推移グラフを生成
  - Excel形式でデータエクスポート

**課題2: 母子手帳システムの成長記録**（過去課題2）
- **問題:** 身長・体重の成長曲線グラフ、予防接種記録の可視化
- **解決:**
  - 成長曲線を折れ線グラフで表示
  - 予防接種スケジュールをガントチャート風に表示
  - 印刷用PDFレポート生成

**課題3: 野球部記録システムの成績管理**（過去課題3）
- **問題:** 打撃成績・防御率・チームスタッツのグラフ化
- **解決:**
  - 打率推移を折れ線グラフで表示
  - チーム別成績を棒グラフで比較
  - シーズンレポートをPDF出力

**共通課題: すべての業務アプリで必要な機能**
- ✅ グラフ生成（Chart.js）
- ✅ PDF出力（印刷用）
- ✅ CSV/Excelエクスポート（データ分析用）
- ✅ レポートテンプレート管理
- ✅ スケジュール実行（週次・月次レポート）

---

## 主な機能

### 📊 グラフ生成
- **Chart.jsラッパー:** Pythonから簡単にグラフ設定を生成
- **対応グラフ:** 折れ線、棒、円、ドーナツ、レーダー、散布図、面グラフ
- **インタラクティブ:** ホバー・ズーム・凡例切り替え

### 📄 PDF出力
- **HTML→PDF変換:** WeasyPrintで印刷レイアウトそのままPDF化
- **テンプレート:** Djangoテンプレートでレイアウト設計
- **グラフ埋め込み:** Chart.jsグラフをPDFに含める

### 📁 データエクスポート
- **CSV出力:** BOM付きUTF-8でExcel対応
- **Excel出力:** openpyxlで基本出力、xlsxwriterでグラフ埋め込み
- **複数形式対応:** 1つのレポートテンプレートから複数形式を生成

### 🏷️ テンプレートタグ
- **{% render_chart %}:** テンプレートから1行でグラフ表示
- **{% render_data_table %}:** データテーブルを簡単表示
- **フィルタ:** format_number, to_json

### 📅 スケジュール実行
- **Cron形式:** 週次・月次レポートを自動生成
- **メール送信:** 生成したレポートを自動配信
- **Celery統合:** 非同期処理で大量データにも対応

## 誰が使うのか？

### 開発者
- **Django開発者:** テンプレートタグで簡単にレポート機能を追加
- **データ分析担当:** CSV/Excelエクスポートでデータ抽出
- **システム管理者:** スケジュール設定で定期レポート配信

### エンドユーザー
- **管理職:** 月次レポートをPDFで受け取り、印刷・配布
- **現場担当者:** リアルタイムで最新データをグラフ表示
- **経営層:** ダッシュボードで重要指標を可視化

## 他のパッケージとの連携

### kits.notifications との連携
```python
# レポート生成完了時に通知
report = report_service.generate_report(template, user)
notification_service.create_notification(
    user=user,
    title="レポート生成完了",
    body=f"{report.title} の生成が完了しました",
    related_object=report,
)
```

### kits.approvals との連携
```python
# 承認完了レポートを自動生成
@receiver(approval_completed)
def generate_approval_report(sender, approval, **kwargs):
    template = ReportTemplate.objects.get(code='approval_summary')
    report_service.generate_report(
        template=template,
        user=approval.requester,
        parameters={'approval_id': approval.id}
    )
```

## 使用例（クイックスタート）

### 1. レポートテンプレート作成
```python
template = ReportTemplate.objects.create(
    code='monthly_overtime',
    name='月次残業レポート',
    model_name='kits.demos.OvertimeRequest',
    chart_config={
        'type': 'bar',
        'x_column': 'department',
        'y_columns': ['total_hours'],
        'title': '部門別残業時間',
    },
    supported_formats=['pdf', 'xlsx'],
)
```

### 2. レポート生成
```python
from kits.reports.services import ReportService

service = ReportService()
report = service.generate_report(
    template=template,
    user=request.user,
    format='pdf',
    parameters={
        'date_from': '2025-10-01',
        'date_to': '2025-10-31',
    }
)

# ダウンロードURL
print(report.file.url)  # /media/reports/2025/10/05/monthly_overtime_..._.pdf
```

### 3. テンプレートでグラフ表示
```django
{% load report_tags %}

<h2>部門別残業時間</h2>
{% render_chart chart_json "overtimeChart" 400 %}
```
