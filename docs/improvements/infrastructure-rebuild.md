# インフラ再構築

ステータス: **次**
優先度: 高

---

## なぜやるか

- 現在 AWS 上のリソースは削除済み（コスト削減のため停止していた）
- Terraform state と実態がずれており、apply で 61 リソースを再作成する必要がある
- Phase 1 のアラーム・ダッシュボード・EventBridge を含む全環境を構築する

## 何をするか

- `terraform plan` で変更内容を確認
- `terraform apply` で全リソースを作成
- 動作確認（ヘルスチェック、ダッシュボード表示、アラーム状態）

## 注意事項

- 61 リソース新規作成（VPC、ALB、EC2、RDS、CloudFront 等すべて）
- 月額コスト: ~$10-15/月（主にパブリック IPv4 課金）
- RDS 作成に 5-10 分かかる
- CloudFront のディストリビューション作成に 10-15 分かかる
- apply 後に EC2 への Docker デプロイが別途必要

## コスト

- パブリック IPv4: ~$10.80/月（EC2 1 個 + ALB 2 AZ）
- RDS: 無料枠内（db.t3.micro 750 時間/月）
- EC2: 無料枠内（t3.micro）
- ALB: 無料枠内（750 時間/月）
- 消費税 10%
