- ブラウザで `http://localhost:8000/admin/` にログイン済み
- 管理画面のトップページが表示されている

1. **「件名テンプレート」** の空欄をクリック

## トラブルシューティング

### エラー: 「この項目は必須です」と表示される

**原因:** 必須項目（\*マークがついている項目）が未入力です。

**解決方法:**

1. 画面を上にスクロールして、赤い文字でエラーが表示されている項目を探す
2. その項目を入力する
3. もう一度「保存」ボタンをクリック

---

### エラー: 「この値はすでに存在します」

**原因:** すでに同じ「テンプレートコード」が存在しています。

**解決方法:**

1. テンプレートコードを別の名前に変更する（例: `welcome_email_2`）
2. または、一覧ページに戻って既存のテンプレートを削除してから再度作成する

---

### 通知タイプのチェックボックスが表示されない

**原因:** Django サーバーが再起動されていない可能性があります。

**解決方法:**

```bash
# ターミナルで実行
docker-compose restart django
```

再起動後、ブラウザをリロード（F5 キー）してください。

## まとめ

以下の手順でテンプレートを作成しました：

1. 管理画面で「通知テンプレート」→「追加」をクリック
2. 基本情報を入力（コード、名前、説明、有効チェック）
3. テンプレート内容を入力（件名、本文）
4. 通知タイプで「メール」にチェック
5. 「保存」ボタンをクリック
6. 成功メッセージを確認
