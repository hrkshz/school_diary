```
  ↓  ↖
denied    return_to_draft（差戻し）
```

#### 提供しているパターン
```python
class DemoRequest(models.Model):
    # 1. 状態管理
    status = FSMField(default='draft')

    # 2. 関係者記録
    created_by = models.ForeignKey(User, ...)
    requester = models.ForeignKey(User, ...)
    approver = models.ForeignKey(User, ...)

    # 3. 履歴管理
    history = HistoricalRecords()

    # 4. 公開メソッド（トランザクション保護付き）
    def submit(self, by=None, reason=None):
        with transaction.atomic():
            self._perform_submit()
            self.save()

    # 5. 内部遷移実装（FSM）
    @transition(field=status, source="draft", target="submitted")
    def _perform_submit(self):
        pass
```
