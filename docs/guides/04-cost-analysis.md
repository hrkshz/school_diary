# AWS コスト分析の実践

## なぜやるか

AWS を使っていて「なぜこの請求が来るのか分からない」は危険。
コストの構造を理解することで、無駄を特定し、最適化の判断ができる。

## 実際に分析した結果

### 請求データ（2025/09 - 2026/02）

| サービス | 累計コスト | 月平均 |
|---------|-----------|--------|
| VPC | $13.74 | ~$4.58 |
| RDS | $6.03 | ~$2.01 |
| Tax | $2.16 | - |
| KMS | $1.30 | - |
| Secrets Manager | $0.52 | - |
| その他 | ≒ $0 | - |
| **合計** | **$23.76** | - |

### 分析 1: VPC $13.74 の正体

VPC 自体は無料。ではなぜ課金されているのか。

**答え: パブリック IPv4 アドレス課金**

2024 年 2 月から、AWS は全てのパブリック IPv4 アドレスに **$0.005/時間**（約 $3.60/月）を課金している。
これは「VPC」の行に計上される。

このプロジェクトでパブリック IP を持つリソース:
- EC2: 1 個（$3.60/月）
- ALB: 2 個（2 AZ に配置されるため、各 AZ に ENI + IP が作られる = $7.20/月）

**11 月の実績 $7.20 はちょうど ALB の 2 IP 分と一致。**

**なぜ ALB に 2 個の IP が必要か:**
ALB は高可用性のため、最低 2 つの AZ にまたがって配置される。
各 AZ に ENI（ネットワークインターフェース）が作られ、それぞれにパブリック IP が割り当てられる。

```
ALB (1台)
├── AZ-a (ap-northeast-1a): ENI + パブリックIP ← $3.60/月
└── AZ-c (ap-northeast-1c): ENI + パブリックIP ← $3.60/月
```

### 分析 2: RDS $6.03 の内訳

RDS db.t3.micro は無料枠内（750 時間/月）。ではなぜ課金されるか。

**推定原因:**
- `storage_encrypted = true` により、暗号化ストレージの I/O コストが発生
- バックアップストレージが無料枠（DB サイズと同量の 20GB）を超えた可能性

### 分析 3: KMS $1.30

**原因:** カスタマーマネージドキー（CMK）の月額課金（$1/月/キー）。
RDS の暗号化で CMK が自動作成された可能性。
2026/01 以降は $0 → リソース削除時に CMK も削除されたと推定。

### 分析 4: Secrets Manager $0.52

**原因:** シークレットの月額課金（$0.40/月/シークレット）。
DB パスワードなどを Secrets Manager に格納していた。
SSM Parameter Store（無料）で代替可能。

## 構造的に避けられないコスト

- パブリック IPv4: ALB + EC2 で ~$10.80/月（構成上必須）
- 消費税 10%

## 最適化で削減できるコスト

- KMS: AWS 管理キーに変更 → $1/月削減
- Secrets Manager: SSM Parameter Store に移行 → $0.40/月削減
- EC2 夜間停止: IP 解放で ~$1.20/月削減
